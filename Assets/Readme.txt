* Notes for Project Creation and Modification - Updated on July 3, 2024


1. Please avoid using the name "Resources" for any folders, objects, scripts, or other files.
- This is because Photon Server uses prefabs in the "Resources" folder for creating objects synchronized between players.

2. Please replace the App ID used to connect to the Photon Server with your own App ID.
- When committing to GitHub, your personal App ID will be included in the commit.
- It would be more efficient for each individual to use their own server rather than having multiple people use the same server.

[Korean]-----

* 프로젝트 제작 및 수정 시 참고 사항 - Updated on July 3, 2024


1. 폴더나 오브젝트, 스크립트 등 모든 파일에 대한 이름으로 "Resources"라는 이름은 사용하지 않길 바랍니다.
- Photon 서버에서 플레이어 간에 연동할 오브젝트 생성은 "Resources" 폴더에 있는 프리팹을 사용하기 때문입니다.

2. 포톤 서버에 연결하는 App ID는 자신의 App ID로 바꿔서 사용해주시기 바랍니다.
- GitHub로 커밋할 때 개인이 설정한 App ID가 그대로 커밋됩니다.
- 다른 사람이 타인 한명의 서버를 그대로 사용하는 것보다, 개인의 서버를 사용하는 것이 더 원활할 것 같습니다.
